# 登陆
将你的正脸照片放置在source目录中，并命名为[名字缩写].xxx  
右键托盘的Dog，打开摄像头进行人脸登陆。  
# Monitor
Dog会记录所有键盘记录和窗口记录，他们在C:\DogTools  
# Breaker
阻断某些窗口，例如历史记录。  
单击主页面，点击下面的“TARGET”，将焦点转移到目标窗口上等待5秒。  
确认窗口信息。
