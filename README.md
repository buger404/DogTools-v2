# 授权
打开`keygener`目录，运行目录下的exe
选择你的U盘，输入你的名字缩写，然后在黑色区域内手写签名
然后点击“授权” 
# 登陆
插入已授权的U盘
# Monitor
Dog会记录所有键盘记录和窗口记录，他们在C:\DogTools  
# Breaker
阻断某些窗口，例如历史记录。  
单击主页面，点击下面的“TARGET”，将焦点转移到目标窗口上等待5秒。  
确认窗口信息。
